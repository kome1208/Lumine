# Lumine

原神プレイヤーのために作られたステータス確認アプリ

## 機能

- リアルタイムノートの確認
- 天然樹脂回復リマインダー
- ゲーム内データの確認
- キャラクター詳細の確認
- デイリーボーナス受け取り（埋め合わせログインも可能）
- 交換コード適用
- 育成素材カレンダー
- etc...
